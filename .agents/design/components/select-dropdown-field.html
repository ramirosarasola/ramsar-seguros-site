§ 5.2

Select / Dropdown Field

Same trigger as Input Field; different reveal. Used for marca (15 options), modelo (100+ options), provincia (24), and any constrained choice. The 8-option threshold triggers built-in search.

BaseIdentical to §5.1 Input Field in dimensions, padding, radius, and all six states (default, hover, focus, filled, error, disabled).
Trigger value displayWhen unselected: placeholder copy ("Seleccioná una marca") · color sand-400. When selected: chosen option label · color sand-900 · weight 400 (not 500 — value is data, not a heading).
Trigger iconchevron-down · icon-sm (16 px) · color sand-500 · position absolute right · 16 px right inset · vertically centered. On panel open: rotates 180° over duration-fast (120 ms) · ease-in-out · color shifts to marine-700.
Options panel · widthEqual to trigger width (anchored). Panel offset 4 px below trigger.
Options panel · surfaceBackground sand-0. Border 1 px sand-200. Radius radius-sm (8 px). Shadow elevation-2. Padding 4 px all sides. Z-index z-40.
Options panel · max-height240 px with internal scroll. Scrollbar 4 px sand-300 thumb on 6 px sand-100 track.
Option itemHeight 40 px. Padding space-2 (8 px) vertical · space-3 (12 px) horizontal. Typography body-m (16 · 400) · color sand-900. Radius radius-sm (8 px) per item (matches hover bg).
Option · hoverBackground marine-50. Text color stays sand-900.
Option · selectedBackground marine-50. Text color marine-700, weight 500. Trailing check icon (icon-sm · color marine-700) right-aligned.
Option · focused (kbd)Same as hover plus 2 px marine-700 focus ring inset 2 px.
Search inputEnabled when option count > 8. Sits at top of panel, separated from options by 1 px sand-100 bottom border. Height 36 px. Leading search icon-sm · color sand-400. Placeholder "Buscar…" (or "Buscar marca…" / "Buscar modelo…" — contextual). Filters options live with a 60 ms debounce. Clears automatically on panel close.
Empty search stateCentered text "No se encontraron resultados" · typography body-s (14 · 400) · color sand-500. Vertical padding space-6 (24 px). Optional helper link below: "Ver todas las marcas" → reset search · typography label-s (11.5 · 500) · color marine-700 · underline 1 px offset 2 px.
AnimationOpen: opacity 0 → 1 + translateY(−4 → 0) · duration-default (220 ms) · ease-out. Close: opacity 1 → 0 + translateY(0 → −4) · duration-fast (120 ms) · ease-in.
KeyboardSpace / Enter / ↓ open the panel and focus first option. ↑ / ↓ move between options. Type-ahead: typing characters when panel is open jumps to the first option matching the typed string. Enter selects the focused option, closes the panel, returns focus to the trigger. Esc closes without selection.
ARIATrigger: role="combobox" aria-expanded aria-controls aria-haspopup="listbox". Panel: role="listbox". Options: role="option" aria-selected. Search input inside panel: role="searchbox".
DoAuto-enable search at 9+ options. Below that, search is friction; above it, search is the only sane interface.
Don'tDon't use a native <select> on mobile "because the OS picker is faster." Native pickers can't surface search and can't render our 2-line option layout (model + year). The custom component is mandatory.
Figma propsstate: closed · open · empty · error — showSearch: boolean — optionCount: 3–20 — selectedIndex: -1 to N — text overrides per option.
