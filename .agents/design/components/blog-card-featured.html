LayoutHorizontal · image-left 55 / content-right 45. Desktop fixed height 320 px (image fills full height). Mobile collapses to vertical — same proportions as §3.3 Grid variant.
Card surfaceBackground sand-0 · border 1 px sand-200 · radius radius-lg (16 px) · shadow elevation-1 default · elevation-2 hover. overflow: hidden.
ImageWidth 55% · height 100% (fills card). Border-radius inherits left corners only (top-left + bottom-left both radius-lg). Object-fit cover. Hover scale 1.03 same as §3.3.
Category badgePositioned absolute · bottom: 16 px · left: 16 px. Same color treatments as §3.3 (6 fixed categories). Slightly larger padding: 4 px / 12 px.
Content area · paddingspace-8 (32 px) all sides. Internal vertical centering (justify-content: center) so short headlines don't pin to the top.
"Destacado" labelAbove the headline. Typography overline (11 · mono · +0.16 em · uppercase · 500) · color marine-500. Bottom margin space-2 (8 px). Reads as section tag, not category.
HeadlineTypography h3 (28 / 34 · 600 · sans) · color sand-900. Max 3 lines · ellipsis. Bottom margin space-3 (12 px).
ExcerptTypography body-m (16 / 26 · 400) · color sand-600. Max 4 lines (one more than §3.3). Bottom margin space-4 (16 px).
Metadata rowIdentical to §3.3 except author/date colors bump one step darker (sand-800 / sand-600) — featured card carries slightly higher visual weight throughout. Bottom margin space-3 (12 px).
"Leer artículo →" linkTypography label-m (13 · 600 · sans) · color marine-700. Trailing arrow-right icon-xs · 4 px gap. Hover: arrow translates 4 px right · underline appears 1 px / 3 px offset. Why explicit link here: the larger card invites scanning; an explicit "read" action grounds the eye at the end of the content block.
States · hoverShadow elevation-1 → elevation-2. Headline → marine-700. Image scales 1.03. Arrow translates. 220 ms · ease-out.
ResponsiveAt md (≤ 1023 px): switches to vertical layout (image top · content bottom) and adopts the §3.3 Grid spec dimensionally. The "Destacado" overline is preserved as the visual differentiator from the grid variant.
AccessibilityEntire card is one <a>. The "Leer artículo →" text is decorative inside that link — it carries aria-hidden="true" so the headline isn't read twice.
DoUse one featured card per page. Two competing featured slots dilute the signal to "nothing here is special".
Don'tDon't make the "Leer artículo →" link a separate tab-stop. It is decorative — the card itself is the destination.
Figma propsSame as §3.3 plus showLeerLink: boolean (default true) — height: fixed · auto.
