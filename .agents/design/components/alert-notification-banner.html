§ 5.5

Alert / Notification Banner

Inline status banner — sits within page content, not floating. Used to surface policy expirations, partial form errors, contextual info, post-action confirmations. Four semantic variants, identical structure.

Variants · infoBG info-light (#E1ECF7) · border-left info-default (#2A5F9E) · border 1 px info @ 30% · icon info-default · title info-dark (#173A66) · text sand-700. Icon: info.
IconSize icon-md (20 px) · stroke 1.75 · color = variant-default. Vertical alignment: top-aligned to title cap-height, not center (when content wraps to many lines, center-alignment drifts the icon).
Content areaFlex 1 (fills available width).
TitleTypography label-m (13 · 600 · sans). Color = variant-dark. Bottom margin space-1 (4 px) when description follows; 0 otherwise.
DescriptionTypography body-s (14 · 400). Color sand-700. Max-width 62 ch. Can contain inline links (color = variant-default · underline 1 px / 3 px offset) and a single trailing action link.
Action linkInline at end of description. Typography label-s (11.5 · 600) · color variant-default · underline. Hover: color darkens to variant-dark.
Dismiss buttonOptional. x icon · icon-sm · color sand-400 · hover sand-700. Position: top-right of container · padding space-3 (12 px) (becomes a 32 × 32 hit-target). On click: alert collapses (height 0 + opacity 0, duration-fast · ease-in).
WidthFull width of parent container. No fixed max-width — caller controls placement.
Accessibilityrole="alert" for time-sensitive (warning, error). role="status" for success and info. aria-live="polite" for status, "assertive" for error. Icons are aria-hidden. Dismiss button has aria-label="Descartar alerta".
DoTop-align the icon to the title's cap-height. When alerts wrap to multiple description lines, a center-aligned icon drifts away from the title — and the icon-title pair is the unit of meaning.
Don'tDon't auto-dismiss alerts. Auto-dismiss belongs to toasts (transient feedback); alerts are persistent context — they live until the user dismisses or the page reloads.
Figma propsvariant: success · warning · error · info — showTitle: boolean — showAction: boolean — dismissible: boolean — text overrides for title · description · action label.
