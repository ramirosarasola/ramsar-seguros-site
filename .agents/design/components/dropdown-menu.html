§ 1.4

Dropdown Menu

Hover-or-focus panel attached to a top-nav link with sub-destinations. Used for "Seguros" (insurer landing pages) and "Vehículos" (brand landing pages). Built for SEO link surfacing as much as for navigation.

Trigger · desktopPointer-enter or focus on a top-nav link with the data-dropdown attribute. 120 ms open-delay (prevents flyover triggers). Click on the trigger itself navigates to the parent index page (e.g. /aseguradoras) — the trigger is also a real destination.
Trigger · touch/mobileDropdowns do not exist on mobile. The same sub-destinations appear as collapsible sections inside §1.3 Mobile Drawer. Touch devices receive direct navigation on tap of the trigger (same href as desktop click).
Panel widthFixed 320 px. Vehículos variant uses 480 px with a 2-column inner grid (15 brands need it).
Panel max-heightmin(480 px, 70vh). If exceeded, internal scroll appears; scrollbar styled to sand-300 thumb.
Panel paddingspace-2 (8 px) all sides. Items themselves carry padding.
Panel backgroundbg-card (sand-0 / white).
Shadow / radiuselevation-2 · radius-md (12 px). All four corners equally rounded.
PositionBelow trigger · left edge of panel aligned to left edge of trigger label · 8 px vertical offset from trigger baseline. If panel would overflow viewport on the right, anchor right-aligned instead (auto-flip).
Item anatomyLeft: optional icon (icon-sm · sand-700). Right: stacked label + description. Right-edge: optional arrow-right icon on hover.
Item · labellabel-l (15 · 500) · color sand-900.
Item · descriptionbody-xs (13 · 400) · color sand-500. Maximum one line; truncate with ellipsis.
Item · paddingspace-3 (12 px) vertical · space-3 (12 px) horizontal. Height result: ≈ 56 px. Items reach the full panel content width.
Item · gapIcon → label cluster: space-3 (12 px). Label → description: 2 px (snug deliberately).
States · defaultBackground transparent · label sand-900 · description sand-500 · icon sand-700.
States · hoverBackground marine-50 · radius radius-sm (8 px) · icon & label → marine-700. Arrow-right icon-sm appears at right edge, translates in 0→4 px. 120 ms · ease-out.
States · focusedSame as hover plus a 2 px marine-700 focus ring outset 2 px inside the panel padding.
States · active / currentIf the user is already on the linked page: background marine-50, label marine-700, plus a 3 px marine-700 rounded indicator pill on the left edge (full item height, 3 px wide, 12 px from left). Display aria-current="page".
Open animationopacity 0→1 + translateY −4 → 0 · transform-origin top · duration-default (220 ms) · ease-out.
Close animationopacity 1→0 + translateY 0 → −4 · duration-fast (120 ms) · ease-in. Faster than open by design (exits should feel decisive).
Keyboard↓ / ↑: move focus between items. Home / End: first / last. Enter / Space: navigate. Esc: close, return focus to trigger. Tab: closes panel and continues to next nav item.
ARIATrigger: aria-haspopup="menu" aria-expanded="true|false" aria-controls="dd-seguros". Panel: role="menu". Items: role="menuitem".
Z-indexz-40 · 40. Below sticky nav (50) so the panel reads as descending from the nav.
DoMake the trigger itself a real link to a parent index page. Dropdowns supplement; they don't replace the index.
Don'tDon't close the dropdown the instant the cursor leaves the trigger. The 240 ms close-delay is non-negotiable — without it, the panel becomes unreachable.
Figma propscolumns: 1 · 2 — itemCount: 4–14 — showDescription: boolean (Vehículos uses false; Seguros uses true) — showIcon: boolean — instance-swap for each menu item.
