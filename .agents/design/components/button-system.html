§ 5.3

Button System

The system's vocabulary of action. Five variants × four sizes = the entire interactive surface. Primary drives conversion, Secondary supports, Ghost recedes, Destructive warns, CTA carries the homepage hero. Anything outside this set is forbidden.

VARIANTS × SIZES
CTA · HERO LOADING DISABLED
Size scale
  • sm: height 32 px · padding 0 space-3 (12 px) · typography label-s (11.5 · 600). Used in toolbars, dense table actions.
  • md: height 40 px · padding 0 space-4 (16 px) · typography label-m (13 · 600). Default size. Used in nav, cards, inline CTAs.
  • lg: height 48 px · padding 0 space-5 (20 px) · typography label-m (13 · 600). Used in modal footers, prominent in-flow actions.
  • xl: height 56 px · padding 0 space-6 (24 px) · typography label-l (15 · 600). Reserved for homepage hero, cotizador entry.
Common specRadius radius-sm (8 px) on all sizes/variants. Display inline-flex · align-items center · gap space-2 (8 px) for icons. Cursor pointer. No text-transform.

Primary · activeBackground marine-900. translateY(0). Shadow elevation-0. Instant (no transition).

Secondary · hoverBackground marine-50. Border + text unchanged.
Secondary · activeBackground marine-100.

Ghost · defaultBackground transparent · border 0 · text marine-700.
Ghost · hoverBackground marine-50. No underline — the bg shift is the entire feedback. Underlines belong to inline text-links, not buttons.
Ghost · activeBackground marine-100.
Destructive · structureSame structure as Primary, error palette throughout. Default bg error-default (#B8341C) · text sand-0. Hover bg error-dark (#7A1F0F). Used only for: cancel policy, delete saved quote, remove account. Always requires a confirmation step — never the first click in a destructive flow.
CTA · the hero buttonBase = Primary at xl size. Additional treatment: colored shadow glow — box-shadow: 0 4px 14px rgba(15, 67, 63, 0.32), 0 2px 4px rgba(15, 67, 63, 0.18). The shadow intensifies on hover: 0 6px 20px rgba(15, 67, 63, 0.4), 0 2px 4px rgba(15, 67, 63, 0.18). No pulse animation, ever. A pulsing CTA in insurance reads as "act before you think" — the opposite of the brand voice. The glow is sufficient.
Icon variants · icon-leftIcon icon-sm (16 px) preceding label · gap space-2 (8 px). Used for affordance ("Volver" with arrow-left).
Icon variants · icon-rightIcon trailing label · same gap. Default for CTA buttons ("Cotizá ahora" + arrow-right).
Icon-onlyEqual padding all sides → square button. Sizes: sm 32×32 · md 40×40 · lg 48×48 · xl 56×56. aria-label required, no exceptions. Used for close, copy, settings cog, etc.
Full-width variantwidth: 100%. Used in mobile CTAs, modal footers, card actions. Available on all variants/sizes via a single boolean property.
AccessibilityAlways a real <button> (or <a> when navigating). Disabled state uses disabled attribute, not aria-disabled (we want it skipped from tab order). Loading state uses aria-busy="true" + the visible spinner. Icon-only buttons require aria-label.
DoTransform labels to the Spanish gerund in loading state ("Cotizando…", "Guardando…"). The verb form tells the user what's happening — generic spinners don't.
Don'tDon't add a pulse animation to the CTA. The glow shadow is the affordance; pulse turns "act" into "panic".
