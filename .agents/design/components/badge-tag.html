§ 5.4

Badge / Tag

Small pill component used for category labels, "Recomendado", "Más popular", status indicators, and filter tags. Pill-shaped (radius-full) always — the rounded form differentiates badges from buttons at a glance.

SIZES & COLORS
★ Recomendado Cobertura nacional Más popular Verificado Cobertura vencida Borrador
SM MD · default LG · with icon ✓ Toyota ✕
Radiusradius-full always. The pill is the badge's signature. Square-cornered "tags" don't exist in this system — they'd be confused with disabled buttons.
Size · smHeight 20 px · padding 0 space-2 (8 px) · typography label-s (10.5 · 600 · sans · +0.01 em). Used: dense table cells, tight metadata rows.
Size · md (default)Height 24 px · padding 0 space-3 (12 px) · typography label-s (11.5 · 600 · +0.02 em). Default. Used: card badges, status indicators.
Size · lgHeight 28 px · padding 0 space-4 (16 px) · typography label-m (12.5 · 600). Used: hero badges ("Aseguradora destacada"), prominent labels.
Color variants · background / text
  • brand-soft: marine-50 / marine-700
  • brand-solid: marine-700 / sand-0 — heavier emphasis
  • accent-soft: amber-300 / amber-700 — "Recomendado", "Más popular" affordances
  • accent-solid: amber-500 / midnight-700 — for badges on dark surfaces
  • success: success-light (rgba success @ 12%) / success-dark
  • warning: warning-light @ 14% / warning-dark
  • error: error-light @ 14% / error-dark
  • info: info-light @ 14% / info-dark
  • neutral: sand-100 / sand-700
Icon supportIcon-left only · icon-xs (12 px) · color inherits text-color · gap space-1 (4 px). Common: ★ on "Recomendado", check on "Verificado".
Non-interactive (informational)No hover state · no cursor change · no tab-stop. Default for category labels, "Recomendado", status indicators. Wrap in <span>, not <button>.
Interactive (filter / removable tag)Hover: bg deepens one shade (e.g. marine-50 → marine-100). Cursor pointer. Focus ring per §05. Removable variant: trailing x icon (icon-xs · color inherits text @ 70% α · hover full opacity). Click on x dismisses the badge (used in filter pill stacks).
AccessibilityInformational badges are <span>. Removable filter badges are wrapped in <button aria-label="Quitar filtro: [label]">. Status badges that convey state (e.g. "Cobertura vencida") use aria-label when their visual meaning requires color recognition — color alone is insufficient for accessibility.
DoUse accent-soft for "Recomendado" / "Más popular". Amber-300 on amber-700 is the warmest, most-trusted recommendation cue the palette can carry without entering CTA territory.
Don'tDon't pair a brand-solid badge with a primary button in the same card. Two solid marine surfaces compete for the eye — badge soft, button solid is the rule.
